## Purpose-aligned identity

The project name is intentionally connected to the actual build direction: **cybersecurity challenge, threat-hunting, and skills platform**. This makes the project easier to defend during evaluation because the name, feature story, and implementation domain all point to the same product idea.

- Keeps the name aligned with cyber hunt and challenge-based security learning.
- Frames the app as a threat-hunting arena with skills progression and security reasoning.
- Adds portfolio-ready positioning for cyber range, CTF, and defensive investigation workflows.

Cybersecurity challenge and threat-hunting platform for skills progression, investigation practice, and defensive reasoning.
